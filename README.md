# Network IP Scanner & Ping Tester

A simple, lightweight Python script to ping a list of IP addresses or DNS servers and find the one with the lowest latency (best ping).

## Features
- Works on Windows, Linux, and macOS.
- Measures latency in milliseconds.
- Automatically sorts and highlights the best IP address.

## How to Run
Navigate to the project directory and run the script:
```bash
python main.py
